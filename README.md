# as7331-rs
Support as7331 driver for esp chips.
## Run example:

```
cargo r_esp32
```
```
cargo r_esp32c2
```
```
cargo r_esp32c3
```
```
cargo r_esp32c6
```
```
cargo r_esp32h2
```
```
cargo r_esp32s2
```
```
cargo r_esp32s3
```
